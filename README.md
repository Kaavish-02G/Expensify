💸 Expensify — Expense Tracker with Analytics

Expensify is a modern expense tracking web application that helps users manage their daily spending, visualize financial habits, and gain insights through interactive analytics and graphs.


🚀 Features

* 📊 Track daily income and expenses
* 📈 Visualize spending with charts and graphs
* 🧾 Categorize transactions (food, travel, bills, etc.)
* 🔍 Detailed analytics for better financial decisions
* 💾 Persistent data storage using MongoDB
* ⚡ Fast and responsive UI


## 🛠️ Tech Stack

* **Frontend:** Next.js, React
* **Backend:** Next.js API Routes
* **Database:** MongoDB
* **Styling:** Tailwind CSS
* **Charts:** Chart.js / Recharts 



📊 Future Improvements

* 🔐 User authentication (login/signup)
* 📅 Monthly/Yearly expense reports
* 📱 Mobile responsive enhancements
* 🔔 Budget alerts and notifications
